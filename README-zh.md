### 核心指标 (Metrics)

| 指标               | 公式                                                                             | 描述                                                         |     
| :----------------- | :------------------------------------------------------------------------------- | :----------------------------------------------------------- | 
| NAV                | $\text{Cash} + \text{Crypto Holdings Value}$                                     | 净资产价值；财库所持有数字资产的总市值。                     |     
| NAV per Share      | $\frac{\text{NAV}}{\text{Shares Outstanding}}$                                   | 每股基础加密资产价值；估值的基准线。                         |     
| mNAV               | $\frac{\text{Market Cap}}{\text{NAV}}$                                           | 市场 NAV 倍数；衡量市场为 DAT 溢价或折价的程度。             |     
| Discount / Premium | $\frac{\text{Market Cap} - \text{NAV}}{\text{NAV}}$                              | 股价与每股 NAV 之间的差距百分比；负值为折价，正值为溢价。    |     
| Crypto per Share   | $\frac{\text{Total Crypto Holdings}}{\text{Shares Outstanding}}$                 | 每股对应的加密资产数量；追踪稀释影响的关键指标。             |

## 资本市场 (Capital Markets)

### 融资工具 (Capital Raising Instruments)

| 标题                      | 描述                                                                               |
| ------------------------- | ---------------------------------------------------------------------------------- |
| SPAC                      | 特殊目的收购公司；通过 IPO 募资并寻找目标公司合并上市的“空白支票”壳公司。          |
| de-SPAC                   | SPAC 与目标公司完成合并，使目标公司成为上市实体的过程。                            |
| RTO                       | 反向收购；私人公司通过收购已上市公司外壳实现上市，而非传统 IPO。                   |
| PIPE                      | 私募股权投资已上市公司；合格投资者直接向公众公司购买折价股票或可转债。             |
| Warrants                  | 认股权证；以固定价格购买股票的权利，常作为 PIPE 或可转债的附加激励。               |
| Pre-funded Warrants       | 预付权证；行权价极低（如 $0.0001），功能等同于股票但可避开受益所有权申报限制。     |
| Convertibles              | 可转换债券；DAT 最主要的融资工具，利息极低或为零，可转换为公司股票。               |
| ATM Offerings             | 在市发行；公司根据市场价格零星增发股票，以实现对稀释节奏的灵活控制。               |
| Preferred Stocks          | 优先股；拥有固定股息及清算优先权的股权，常用于结构化的 DAT 融资。                  |
| Secured Credit Facilities | 有担保信贷额度；以加密资产为抵押的贷款，在保留资产潜在上涨空间的同时获得运营资金。 |

### 监管文件 (Filing)

| 标题                    | 描述                                                                             |
| :---------------------- | :------------------------------------------------------------------------------- |
| Shelf Registration      | 储架注册（通常为 S-3）；允许公司预先提交注册文件，分批多次发行证券。             |
| WKSI                    | 知名成熟发行人；市值超过 7 亿美元的大型上市公司，可享受 SEC 自动生效的储架注册。 |
| Auto Shelf Registration | 自动储架注册；WKSI 提交后立即生效的注册方式，提供极致的融资灵活性。              |
| Resale Registration     | 转售注册；为现有持有者（如 PIPE 投资者）登记其受限证券以便在二级市场公开抛售。   |
| S-1                     | 本地首次公开募资或首次证券发行的全面登记表。                                     |
| S-3                     | 成熟发行人的简略登记表；用于储架发行、ATM 及后续股/债融资。                      |
| F-1 / F-3               | 外国私人发行人的专属登记表（对应 S-1/S-3）；例如 Metaplanet 在美股发行时使用。   |
| 8-K                     | 重大事项即时公告报告；涵盖加密资产购买、财库战略变更、可转债发行等。             |
| DEF 14A                 | 股东大会委托书；包含财库战略授权、股票发行上限变动等投票事项。                   |
| SC 13D / 13G            | 受益所有权披露；当投资者持有上市公司股份累计达到或超过 5% 时必须申报。           |
