### Metrics

| Metric            | Formula                                                                          | Description                                                                                            |     
| :----------------- | :------------------------------------------------------------------------------- | :----------------------------------------------------------------------------------------------------- | 
| NAV                | $\text{Cash} + \text{Crypto Holding Value}$                                     | Net Asset Value; total market value of the treasury's digital asset holdings.                          |     
| NAV per Share      | $\frac{\text{NAV}}{\text{Shares Outstanding}}$                                   | Per-share value of the underlying crypto holdings; the baseline for valuation.                         |     
| mNAV               | $\frac{\text{Market Cap}}{\text{NAV}}$                                           | Market NAV multiple; measures how much premium or discount the market assigns to the treasury wrapper. |     
| Discount / Premium | $\frac{\text{Market Cap} - \text{NAV}}{\text{NAV}}$                              | Percentage gap between share price and NAV per share; negative = discount, positive = premium.         |     
| Crypto per Share   | $\frac{\text{Total Crypto Holdings}}{\text{Shares Outstanding}}$                 | Amount of crypto backing each share; key metric for tracking dilution impact.                          |

## Capital Markets

### Capital Raising Instruments

| Title                     | Description                                                                                                                                                     |
| ------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| SPAC                      | Special Purpose Acquisition Company; a blank-check shell that raises IPO capital then merges with a target to take it public.                                   |
| de-SPAC                   | The closing merger between a SPAC and its target company, converting the target into a publicly traded entity.                                                  |
| RTO                       | Reverse Takeover; a private company acquires a public shell to gain a listing without a traditional IPO.                                                        |
| PIPE                      | Private Investment in Public Equity; accredited investors buy discounted stock or convertibles directly from a public company.                                  |
| Warrants                  | Rights to purchase shares at a fixed price; often bundled with PIPEs or convertibles as an investor sweetener.                                                  |
| Pre-funded Warrants       | Warrants priced near the share price with a nominal exercise price (e.g. $0.0001); functionally equivalent to shares but avoid beneficial ownership thresholds. |
| Convertibles              | Zero-coupon or low-coupon notes/debentures convertible into equity; primary funding tool for DATs.                                                              |
| ATM Offerings             | At-the-market equity offerings that sell shares incrementally at prevailing market prices for controlled dilution.                                              |
| Preferred Stocks          | Senior equity with fixed dividends and liquidation priority; often used in structured DAT financings.                                                           |
| Secured Credit Facilities | Crypto-collateralized loans (e.g. via Coinbase Prime, Galaxy) providing capital while retaining asset upside.                                                   |

### Filing

| Title                   | Description                                                                                                                   |
| :---------------------- | :---------------------------------------------------------------------------------------------------------------------------- |
| Shelf Registration      | A pre-filed registration (usually S-3) that lets a company offer securities over time without a new filing for each sale.     |
| WKSI                    | Well-Known Seasoned Issuer; large public companies ($700M+ float) eligible for automatic shelf registration with the SEC.     |
| Auto Shelf Registration | Shelf registration available to WKSIs that becomes effective immediately upon filing; maximum flexibility for capital raises. |
| Resale Registration     | Registration filed to allow existing holders (e.g. PIPE investors) to resell restricted securities on the public market.      |
| S-1                     | Initial registration statement for domestic IPOs or first-time public offerings of securities.                                |
| S-3                     | Short-form registration for seasoned issuers; used for shelf offerings, ATMs, and follow-on equity/debt raises.               |
| F-1 / F-3               | Foreign private issuer equivalents of S-1 and S-3; used by non-US DATs (e.g. Metaplanet) listing on US exchanges.             |
| 8-K                     | Current report for material events — crypto acquisitions, treasury strategy changes, convertible note offerings, etc.         |
| DEF 14A                 | Proxy statement for shareholder meetings; includes votes on treasury strategy authorization and share issuance caps.          |
| SC 13D / 13G            | Beneficial ownership disclosure required when an investor acquires ≥5% of a public company's shares.                          |
